## Adding a Skill

1. Create a directory: `SKILLS/<skill-name>/`
2. Add a `SKILL.md` file following our [conventions](SKILL-AUTHORING.md#skill-frontmatter)
3. Update this file with your skill entry
4. Submit as a Pull Request ([how](CONTRIBUTING.md))
